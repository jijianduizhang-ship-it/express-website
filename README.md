# 速达快递官网

一个现代化的快递服务官网前端页面。

## 🚀 在线预览

访问地址：https://pengfei-express.github.io/express-website/

## ✨ 功能特点

- 📱 响应式设计，完美适配手机和电脑
- 🎨 现代化 UI 设计，使用 Tailwind CSS
- 🔍 运单查询功能
- 💰 运费估算器
- 📍 网点查询
- 📦 物流跟踪时间线

## 🛠️ 技术栈

- HTML5
- Tailwind CSS (CDN)
- Font Awesome 图标
- Vanilla JavaScript

## 📁 项目结构

```
express-website/
├── index.html      # 主页面
├── script.js       # JavaScript 交互逻辑
└── README.md       # 项目说明
```

## 🚀 本地运行

直接在浏览器中打开 `index.html` 文件即可预览。

## 📄 页面说明

### 首页 (Home)
- 公司介绍和数据统计
- 快速功能入口

### 寄件服务 (Services)
- 当日达、次日达服务
- 上门取件
- 大件物流

### 运单查询 (Tracking)
- 输入运单号查询物流状态
- 可视化时间线展示

### 运费估算 (Calculator)
- 根据省份和重量计算运费
- 支持多种服务类型

### 网点查询 (Network)
- 搜索附近服务网点
- 显示地址、电话、营业时间

### 关于我们 (About)
- 公司介绍
- 数据统计展示
